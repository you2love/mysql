# MySQL 学习教程网站

一个纯静态的 MySQL 学习教程网站，包含从基础到高级的完整学习内容。

## 项目简介

这是一个现代化的 MySQL 教程网站，提供结构化的学习路径和实用的代码示例。网站采用响应式设计，帮助用户从入门到精通 MySQL 数据库。

### 添加新章节

在 `index.html` 中添加新的 `<section>` 元素：

```html
<section id="your-section" class="section">
    <h2>你的章节标题</h2>
    <div class="content">
        <!-- 内容 -->
    </div>
</section>
```

同时在导航栏中添加对应的链接：

```html

```

## 常见问题

### Q: 如何连接到本地 MySQL 服务器？
A: 使用命令 `mysql -u root -p` 连接到本地服务器。

### Q: 忘记 MySQL root 密码怎么办？
A: 参考 [MySQL 官方文档](https://dev.mysql.com/doc/refman/8.0/en/resetting-permissions.html) 重置密码。

### Q: 如何优化查询性能？
A: 创建合适的索引、避免 SELECT *、使用 EXPLAIN 分析查询计划。

### Q: 本网站支持哪些 MySQL 版本？
A: 本教程基于 MySQL 9.x 版本编写，包含最新特性，但也适用于 MySQL 8.x。
