📖 什么是 MySQL？

MySQL 是世界上最流行的开源关系型数据库管理系统，广泛应用于 Web 应用、企业系统和云计算平台。它以其卓越的性能、可靠性和易用性而闻名。

30+ 年历史

⚡ 快速开始

1️⃣

安装 MySQL

根据你的操作系统（Mac/Linux/Windows）安装 MySQL 9.x 版本

2️⃣

连接数据库

mysql -u root -p
3️⃣

创建第一个表

CREATE DATABASE mydb;
USE mydb;
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(50),
    email VARCHAR(100)
);
